Общая информация отборочного тура

Цель отборочного тура:

  • дать возможность командам самостоятельно оценить уровень подготовки к участию в Чемпионате;
  • отобрать команды, которые смогут бороться во втором отборочном туре (полуфинале) за выход в финал Чемпионата.

Начало отборочного тура: 19 марта 2024 года в 16:00.
Окончание отборочного тура: 25 марта 2024 года в 05:00.

Четвертьфинал является обязательным отборочным этапом для команд БГУИР и школьников. Остальные команды по желанию также могут принять участие в данном туре.

Команды, решившие не менее половины из предложенных задач, проходят во второй отборочный тур (полуфинал соревнований).

Доступ к системе тестирования:

Условия задач доступны в разделе задачи и в тестирующей системе.

Для участия в отборочном этапе необходимо, чтобы регистрация была подтверждена. Список зарегистрированных команд и их статус можно посмотреть по ссылке

Логины и пароли для доступа к тестирующей системе указаны в личном кабинете.

Общая информация полуфинала

Полуфинал XII Международного открытого чемпионата БГУИР по программированию пройдет 4 апреля с 14:00 до 19:00 по минскому времени. Для входа в тестирующую систему будут использоваться новые логины и пароли (будут разосланы по почте перед началом контеста).

Напомним, что данный этап является обязательным в том числе для команд из других вузов. Список участников полуфинала можно найти по ссылке.

По правилам наших соревнований второй отборочный этап (полуфинал) команды должны решать в своих учреждениях образования, однако окончательное решение о месте проведения контеста остается за тренером команды. Участвовать в полуфинале можно в учебных аудиториях БГУИР только при предварительном уведомлении организационного комитета по средства письма на электронный адрес acm@bsuir.by в срок до 27.03.2023 13:00.

Вот простые правила fair-play, которые необходимо соблюдать во время полуфинального контеста:

  • Команда должна использовать только один компьютер для решения контеста. Исключение: если участники команды решают контест удаленно друг от друга, то для написания кода разрешается использовать ТОЛЬКО ОДНУ клавиатуру в один момент времени; допускается использование чатов и аудио и видео коммуникации внутри команды.
  • Допускается чтение условий (и ничего кроме этого) с разных устройств.
  • Разрешается общаться только с членами своей команды и через тестирующую систему с представителями жюри.
  • Запрещается обсуждать идеи и обмениваться условиями и решениями задач вне команды. Также запрещено публиковать или делиться любой информацией, имеющей отношение к задачам, до окончания полуфинала.
  • Разрешено использовать предварительно написанный участниками команды код или любой другой код, который находился в публичном доступе до начала полуфинала.
  • Разрешается пользоваться справочными интернет-ресурсами.

Команды, которые нарушат правила, будут дисквалифицированы. В случае возникновения спорных ситуаций бремя доказательства соблюдения правил полностью ложится на участников.
